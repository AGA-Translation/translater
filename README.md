If you want to contribute code, please check the [Contribution Guide](https://github.com/getgridea/gridea/wiki/%E8%B4%A1%E7%8C%AE%E6%8C%87%E5%8D%97) in advance.

$ cd gridea
$ yarn
$ yarn electron:serve
$ yarn electron:build
